# RESTful

* REpresentational State Transfer

## HTTP (CRUD)

* Create => POST
* Read => GET
* Update => PUT
* Delete => DELETE

## Example

* POST
    `/apis/restaurants`

* GET
    `/apis/restaurants`

    `/apis/restaurants/1`

* PUT
    `/apis/restaurants/1`

* DELETE
    `/apis/restaurants/1`
